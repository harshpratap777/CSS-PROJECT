# 💻 Cashify Clone

This project is a **static clone** of the [Cashify](https://www.cashify.in/) website, built using **HTML** and **CSS**. It replicates the **visual design and layout** of key sections of the original platform, showcasing front-end development skills and attention to UI detail.
