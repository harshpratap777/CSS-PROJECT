💻 Cashify Clone
This project is a static clone of the Cashify website, built using HTML and CSS. It replicates the visual design and layout of key sections of the original platform, showcasing front-end development skills and attention to UI detail.

🛠️ Technologies Used
HTML5 – Structuring the content and layout

CSS3 – Styling elements, designing responsive layouts, and adding interactivity

✨ Features
The clone includes several key UI elements from the original Cashify website:

✅ Responsive navigation bar

✅ Hero section with prominent call-to-action

✅ Product listing section styled with cards

✅ Footer with useful links and social media icons

✅ Basic interactive elements such as hover effects and clean form layouts

Note: This project is purely front-end and does not include any backend logic or API integration.
